# vodafone

Vodafone Group plc — UK-headquartered global telecommunications operator (Newbury, England) serving 279M mobile customers and 5M business customers across 15 country markets and 40+ partner markets in Europe and Africa, with 215M+ IoT connections.

This catalog entry tracks Vodafone's commercial API surface published via the **Vodafone Developer Marketplace** ([developer.vodafone.com](https://developer.vodafone.com)) — 27+ APIs across Identity, Mobile, Analytics, Boost/QoS, and Partner categories, including the CAMARA-aligned Open Gateway APIs (Number Verification, SIM Swap, Match, Number Recycling, Quality on Demand) that Vodafone exposes as a premier member of the CAMARA alliance and the GSMA Open Gateway initiative.
